Local-only encryption

OpenPGP Online

Encrypt a message for one or more recipients without sending it to an application server.

Understand the trust boundary

Encryption happens in this browser, but you must still trust the page, device, browser extensions, and hosting path. Public-key encryption does not require your private key.

Signing (optional)

Signing proves who created the message, but requires a private key.

Private-key warning: only continue if you have independently verified this application. Prefer a dedicated, low-value signing key.
